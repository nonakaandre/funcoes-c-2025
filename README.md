# Funções em C - 1º termo 2025

Repositório com exercícios e práticas focadas no uso de **funções na linguagem C**.

## 📌 Objetivo

Praticar e consolidar conceitos fundamentais de programação em C, com foco em:

- Criação de funções
- Organização de código
- Reutilização de lógica
- Separação de responsabilidades

## 🧠 Contexto

As funções são um dos pilares da programação, permitindo dividir o código em partes menores e reutilizáveis.

Este repositório reúne exercícios voltados para fixação desses conceitos.

## 🛠️ Tecnologias utilizadas

- Linguagem C
- Compiladores padrão (GCC, etc.)

## 📂 Estrutura

Os arquivos representam exercícios ou exemplos práticos.
